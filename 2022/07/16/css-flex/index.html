css flex 深入理解

flex item右靠齐 下靠齐

注意 如果item右靠齐 需要给flex 容器写width, 同理 下靠齐 需要给flex容器写height

flex 利用好width flex-wrap

如果是这种布局

flex: none | auto | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ]

  • flex-grow 指定容器有剩余空间的时候是否分配 【默认是0 不分配】
  • flex-shrink 指定容器空间不足是否收缩 [默认是1 收缩]
  • flex-basis item占据的分配数量[默认是auto]
    flex-shrink这个属性可有可无，也可以是两个值

    IE和Chrome的默认值不一样，是实际开发写三个值

flex: 1

代表

  • flex-grow: 1
  • flex-shrink: 1
  • flex-basis: 0% 不是auto （不管是否设置宽度，flex-basis都是0，所有item平分空间）

flex: 100px

代表

  • flex-grow: 1
  • flex-shrink: 1
  • flex-basis: 100px

flex: 1 2

代表

  • flex-grow: 1
  • flex-shrink: 2
  • flex-basis: 0%

flex: 1 100px

代表

  • flex-grow: 1
  • flex-shrink: 1
  • flex-basis: 100px

flex: 1 2 50%

代表

  • flex-grow: 1
  • flex-shrink: 2
  • flex-basis: 50%

flex: initial === flex: 0 1 auto flex的默认值

  • flex-grow: 0 // 不会增长变大占据容器的剩余空间
  • flex-shrink: 1 // 会收缩变小以适应容器
  • flex-basis: auto // 尺寸根据自身宽高属性进行调整

flex: auto === flex: 1 1 auto

  • flex-grow: 1 //会增长变大占据容器的剩余空间
  • flex-shrink: 1 // 会收缩变小以适应容器
  • flex-basis: auto // 尺寸根据自身宽高属性进行调整

flex: none === flex: 0 0 auto

  • flex-grow: 0 // 不会增长变大占据容器的剩余空间
  • flex-shrink:0 // 不会收缩变小以适应容器
  • flex-basis: auto // 尺寸根据自身宽高属性进行调整
